Data.Analytics.Key.Interleaving

Description

This provides a generalized key interleaving system for variable-length keys. For fixed-length keys this looks like the Morton or Z-order.

data Interleaving a Source

This provides a priority queue with a fixed schedule that can be used to obtain an interleaved sequence of bits from multiple sources. Use the monoid to fairly interleave breaking ties to the left.
